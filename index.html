PAC-MAN

Classic Game Implementation in Java

Use arrow keys to move | Press any key to restart after game over

Game Rules

  • Controls: Arrow keys (↑↓←→)
  • Food pellets: 10 points each
  • Cherries: 50 points each
  • Lives: Start with 3 lives
  • Teleport: Tunnel through screen edges

Implementation

  • Language: Java
  • UI Framework: Swing (original)
  • Compilation: TeaVM to WebAssembly
  • Build Tool: Maven
  • Rendering: HTML5 Canvas

Features

  • Object-oriented game architecture
  • Collision detection system
  • Random Movement for Ghosts
  • Score tracking and lives system
  • Browser-based deployment
